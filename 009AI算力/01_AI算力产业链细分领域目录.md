# AI 算力产业链细分领域目录

> 本目录基于图片中的框架整理，后续可围绕每个领域继续扩展为独立研究笔记。

## 1. 算力核心硬件

### AI 芯片

作用：AI 训练和推理的核心计算单元，是算力产业链的源头。

图片公司：海光信息、寒武纪、沐曦股份、摩尔线程。

研究重点：

1. GPU、ASIC、NPU 等技术路线。
2. 训练与推理需求差异。
3. 国产替代空间和生态壁垒。
4. 客户验证、量产能力、软件生态。

### AI 服务器

作用：承载 AI 芯片、存储、网络和散热系统，是算力基础设施的整机载体。

图片公司：工业富联、浪潮信息、紫光股份、华勤技术。

研究重点：

1. AI 服务器与通用服务器区别。
2. GPU 服务器、液冷服务器、高密度服务器。
3. 代工模式、品牌模式、客户集中度。
4. 订单、毛利率和供应链稳定性。

### 存储芯片

作用：为 AI 训练和推理提供高带宽、高容量数据读写能力。

图片公司：德明利、兆易创新、佰维存储、江波龙。

研究重点：

1. HBM、DRAM、NAND、企业级 SSD。
2. 存储价格周期。
3. AI 服务器对高性能存储的拉动。
4. 国产存储产业链位置。

## 2. 高速互联与光通信

### CPO

作用：将光模块与交换芯片更深度集成，降低高速互联中的功耗和延迟。

图片公司：中际旭创、新易盛、天孚通信、华工科技。

研究重点：

1. CPO 与可插拔光模块的区别。
2. 800G、1.6T 速率升级节奏。
3. 海外云厂商资本开支影响。
4. 光模块龙头的盈利弹性。

### OCS

作用：光电交换系统中的光路调度技术，用于提升数据中心网络效率。

图片公司：腾景科技、福晶科技、光库科技、德科立。

研究重点：

1. OCS 的应用场景和渗透率。
2. 与传统交换机网络的关系。
3. 光学器件、晶体、激光器等上游环节。
4. 技术成熟度和商业化节奏。

### 光芯片

作用：光模块的核心器件，决定高速光通信性能。

图片公司：源杰科技、仕佳光子、光迅科技、长光华芯。

研究重点：

1. 激光器、探测器、调制器等器件。
2. 高端光芯片国产化率。
3. 速率升级对芯片价值量的影响。
4. 研发壁垒、良率和客户认证。

### 光纤光缆

作用：数据中心内外部连接和通信网络传输基础设施。

图片公司：长飞光纤、亨通光电、中天科技、烽火通信。

研究重点：

1. 光纤光缆需求与数据中心建设关系。
2. 行业价格周期。
3. 海缆、电力通信、数据中心网络场景差异。
4. 产能和海外业务。

### 光模块设备

作用：光模块生产制造和测试环节的设备支持。

图片公司：罗博特科、科瑞技术、猎奇智能、镭神技术。

研究重点：

1. 光模块自动化设备需求。
2. 高速光模块测试与封装设备。
3. 下游扩产节奏。
4. 设备订单兑现周期。

## 3. 连接、载板与材料

### PCB

作用：AI 服务器和交换机中的关键电子互联载体，高速高密度趋势提升价值量。

图片公司：胜宏科技、东山精密、深南电路、沪电股份。

研究重点：

1. HDI、高多层板、高速板。
2. AI 服务器和高速交换机对 PCB 的需求变化。
3. 材料、良率、客户认证。
4. 产能扩张和盈利弹性。

### CPC

作用：芯片封装、连接与散热相关材料或结构件环节，需结合具体公司产品进一步确认。

图片公司：立讯精密、兆龙互连、沃尔核材、鼎通科技。

研究重点：

1. 产品在 AI 服务器或高速互联中的具体位置。
2. 单机价值量和客户结构。
3. 技术路线变化带来的需求弹性。
4. 与铜缆、连接器、散热材料的关联。

### 铜缆

作用：短距离高速互联方案，在部分 AI 服务器和数据中心内部连接中具备成本和功耗优势。

图片公司：诺德股份、铜冠铜箔、嘉元科技、德福科技。

研究重点：

1. 铜互联与光互联的适用场景。
2. 高速铜缆、铜箔、连接器材料需求。
3. 速率升级后铜方案的生命周期。
4. 价格、加工费和产能周期。

### 树脂

作用：PCB、封装材料和电子材料中的关键基础材料。

图片公司：东材科技、圣泉集团、美联新材、宏昌电子。

研究重点：

1. 高速高频材料需求。
2. Low Dk、Low Df 材料性能。
3. 国产替代和客户认证。
4. 原材料价格与毛利率。

### 电子布

作用：覆铜板和 PCB 的上游材料，影响高速板性能和成本。

图片公司：宏和科技、中国巨石、中材科技、国际复材。

研究重点：

1. 电子级玻纤布供需。
2. 高端电子布国产化。
3. PCB 景气度传导。
4. 价格周期和产能扩张。

## 4. 数据中心与冷却系统

### AIDC

作用：AI 数据中心，是算力部署、供电、制冷、运维和网络的综合基础设施。

图片公司：润泽科技、网宿科技、光环新网、数据港。

研究重点：

1. AIDC 与传统 IDC 的差异。
2. 上架率、机柜功率密度、PUE。
3. 客户结构和长期合同。
4. 电力指标和区域资源约束。

### 液冷

作用：解决高功率 AI 服务器散热问题，提高数据中心能效。

图片公司：英维克、高澜股份、中科曙光、申菱环境。

研究重点：

1. 冷板式、浸没式、喷淋式技术路线。
2. 液冷渗透率和数据中心功率密度。
3. 单机柜价值量。
4. 客户验证、规模交付、售后运维。

### 算力租赁

作用：通过租赁 GPU 或算力资源实现商业化，是算力资产变现方式之一。

图片公司：利通电子、协创数据、宏景科技、优刻得。

研究重点：

1. 算力租赁商业模式和定价方式。
2. GPU 资产成本、折旧、利用率。
3. 客户质量和合同稳定性。
4. 现金流、负债和回本周期。

## 5. 电力与能源基础设施

### HVDC

作用：高压直流供电系统，提升数据中心供电效率和稳定性。

图片公司：中恒电气、圣阳股份、欧陆通、麦格米特。

研究重点：

1. 数据中心供电架构变化。
2. HVDC 与 UPS 的关系。
3. 单机柜功率密度提升带来的需求。
4. 客户认证和项目交付能力。

### 燃气轮机

作用：为大型数据中心或园区提供分布式能源、备用电源或综合能源方案。

图片公司：杰瑞股份、联德股份、应流股份、东方电气。

研究重点：

1. 数据中心电力瓶颈。
2. 燃机在分布式能源中的定位。
3. 订单周期、交付周期和国产替代。
4. 与电网、储能、绿电的协同。

### SST

作用：固态变压器等新型电力电子设备，用于提升电力系统灵活性和效率。

图片公司：四方股份、中国西电、伊戈尔、金盘科技。

研究重点：

1. SST 在数据中心、电网和新能源中的应用。
2. 技术成熟度和商业化难度。
3. 电力电子能力和客户场景。
4. 订单验证与示范项目。

### 算电协同

作用：将算力调度与电力资源协同，解决 AI 数据中心能耗和电力约束问题。

图片公司：豫能控股、协鑫能科、南网数字、韶能股份。

研究重点：

1. 算力布局与电力资源匹配。
2. 绿电、储能、源网荷储一体化。
3. 政策、牌照和区域资源。
4. 商业模式可持续性。

# AI 算力细分领域研究提示词

> 用法：把 `{细分领域}` 替换为具体方向，例如 CPO、AI 服务器、液冷、HVDC、AIDC、PCB、光芯片等。

> 保存规则：生成后的研究内容统一保存到 `009AI算力/{细分领域}/` 目录下。若目录不存在，请先创建目录。建议文件名使用 `YYYY-MM-DD_{细分领域}_细分领域研究.md`、`YYYY-MM-DD_{细分领域}_深度研究.md`、`YYYY-MM-DD_{细分领域}_景气度比较.md` 等格式，确保后续公司研究、财报验证和跟踪数据都能按细分领域归档。

## 1. 标准版提示词

```text
请围绕 AI 算力产业链中的「{细分领域}」生成一份面向二级市场投资研究的学习内容和分析报告。

我的目标不是泛泛科普，而是建立可用于股票投资判断的研究框架。请按照以下结构输出：

1. 一句话定义：这个细分领域是什么，解决 AI 算力中的什么问题。
2. 产业链位置：它位于 AI 算力产业链的哪个环节，上游、下游分别是谁。
3. 需求驱动：需求来自 AI 芯片放量、AI 服务器升级、数据中心建设、网络速率升级、电力约束、国产替代，还是其他因素。
4. 技术路线：主流技术路线、关键技术指标、未来演进方向，以及可能发生替代的风险。
5. 商业模式：主要产品、收入模式、定价方式、客户类型和订单周期。
6. 行业空间：从量、价、渗透率、单机价值量或资本开支角度分析市场空间。
7. 竞争格局：国内外主要玩家、龙头公司、进入壁垒、竞争强度、国产替代情况。
8. 财务特征：这个行业通常体现为什么样的毛利率、净利率、资本开支、存货、应收账款和现金流特征。
9. 投资逻辑：该领域在二级市场中通常被如何定价，什么阶段容易出现估值扩张，什么阶段容易杀估值。
10. 重点公司：列出 A 股、港股、美股中与该领域相关的重点公司，并说明它们的产业链位置。
11. 关键跟踪指标：列出后续需要持续跟踪的数据、公告、订单、价格、政策、技术节点和客户动向。
12. 主要风险：包括需求、技术、价格、竞争、政策、客户集中、财务质量和估值风险。
13. 研究结论：用“高景气逻辑、业绩兑现路径、核心风险、适合重点跟踪的公司”四个维度总结。

要求：
- 输出要专业、结构化，适合保存为投资研究笔记。
- 生成后请保存到 `009AI算力/{细分领域}/YYYY-MM-DD_{细分领域}_细分领域研究.md`；如果目录不存在，请先创建。
- 不要给直接买卖建议。
- 如果涉及最新数据，请标注数据日期和来源。
- 对不确定的信息要明确说明，不要编造。

```

## 2. 深度研究版提示词

```text
请把「{细分领域}」作为 AI 算力产业链中的一个投资研究专题，生成一份深度研究报告。

请从“产业逻辑、技术逻辑、商业逻辑、财务逻辑、估值逻辑、市场预期差”六个层面展开。

报告结构如下：

一、核心结论
1. 这个领域是否处于高景气阶段。
2. 景气度来自真实需求、技术升级、国产替代，还是主题催化。
3. 哪类公司最可能受益。
4. 目前研究中最大的不确定性是什么。

二、产业链拆解
1. 上游材料、设备和核心零部件。
2. 中游制造、集成或解决方案。
3. 下游客户和应用场景。
4. 与 AI 芯片、AI 服务器、数据中心、电力、光通信等环节的关系。

三、技术路线
1. 当前主流技术方案。
2. 核心性能指标。
3. 未来 1-3 年技术演进方向。
4. 技术替代风险和受益方向。

四、市场空间测算
请给出可复用的测算公式，例如：
市场空间 = AI 服务器出货量 x 单台价值量 x 渗透率
或：
市场空间 = 数据中心资本开支 x 对应环节价值占比

请同时说明关键假设、敏感性变量和可能误差。

五、竞争格局
1. 国内外主要公司。
2. 各公司的产品、客户、技术和产能差异。
3. 进入壁垒。
4. 行业集中度和价格竞争风险。

六、财务与估值
1. 典型财务特征。
2. 收入增长、毛利率、净利率、现金流的观察重点。
3. 适合使用 PE、PS、PEG、EV/EBITDA 或订单市值比中的哪类估值方法。
4. 二级市场估值扩张和收缩的触发条件。

七、A 股映射
请列出相关公司，并按以下维度整理：
公司名称、产业链位置、核心产品、客户结构、业绩弹性、主要风险、需要重点验证的问题。

八、跟踪清单
请输出一个表格，包含：
跟踪指标、指标含义、数据来源、更新频率、对投资判断的影响。

九、结论
请用投资研究语言总结：
1. 这个领域的核心矛盾。
2. 最值得关注的变化。
3. 哪些信号代表逻辑正在兑现。
4. 哪些信号代表逻辑可能被证伪。

要求：
- 面向二级市场投资研究，不要写成科普文章。
- 生成后请保存到 `009AI算力/{细分领域}/YYYY-MM-DD_{细分领域}_深度研究.md`；如果目录不存在，请先创建。
- 不给直接买卖建议。
- 需要区分事实、推论和假设。
- 如使用最新数据，请提供数据日期和来源。
```

```text
请回答：
1. 它是什么？

8. 后续我应该继续学习哪些关键词？

要求语言清晰，但不要过度简化。请保留投资研究需要的专业判断。

请将生成内容保存到 `009AI算力/{细分领域}/YYYY-MM-DD_{细分领域}_快速入门.md`；如果目录不存在，请先创建。
```
